# **App Name**: BrandBloom

## Core Features:

- Hero Section: Displays the brand's vision statement with a prominent call-to-action button. Enables smooth scrolling to other sections.
- Testimonials Section: Displays customer testimonials to build trust.
- Founder's Section: Introduces the founder with a personal message and photo.
- Events Gallery: Showcases past events using a responsive photo grid.
- Contact Form: Allows visitors to send messages with an automated success message upon submission.
- Social Media Links: Links to the brand's social media profiles in the footer.
- Announcement Banner: Displays temporary announcements with a dismissible option.

## Style Guidelines:

- Primary color: Sunny Yellow (#FFDA63) for vibrancy and happiness.
- Background color: Light beige (#F5F5DC), a very lightly saturated tint of the primary hue, for a clean, positive feel.
- Accent color: Bright Teal (#7FFFD4) to complement yellow and provide visual interest.
- Body and headline font: 'PT Sans', a humanist sans-serif with a friendly, modern, and readable style.
- Single-page layout with a fixed/sticky header for easy navigation. All sections are present on the homepage for continuous scrolling.
- Smooth scrolling enabled globally for animated navigation to different sections.
- Use modern, simple icons for social media links in the footer.
